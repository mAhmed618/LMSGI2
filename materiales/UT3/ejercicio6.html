Mi primera página web en HTML

En HTML, las etiquetas básicas son elementos fundamentales para la estructura de una página web. Utilizamos la etiqueta <!DOCTYPE html> para declarar la versión de HTML que estamos utilizando. El elemento <html> es la envoltura principal de todo el contenido HTML, y dentro de él, encontramos <head> y <body> que contienen información del documento y el contenido visible, respectivamente.

Los atributos son configuraciones adicionales que podemos agregar a nuestras etiquetas. Por ejemplo, el atributo lang="es" en la etiqueta <html> define el idioma del documento. También utilizamos charset="UTF-8" para especificar el conjunto de caracteres. La etiqueta <title> dentro de <head> nos permite definir el título de la página que se muestra en el navegador.
